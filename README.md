# 🎁 Marvel Chests — 功能規格書

> **版本：** v2.0 ｜ **建立日期：** 2026-05-13 ｜ **最後更新：** 2026-05-18 ｜ **狀態：** 草稿（待企劃確認）

Marvel Chests 是一個與老虎機轉輪綁定的**常駐**寶箱獎勵系統。玩家於轉輪單次贏分達到押注金額的 N 倍以上時，獲得不同稀有度的寶箱，每個寶箱需等待倒數時間後才能開啟，並可使用寶石縮短等待。整體機制旨在提升每日回訪率（DAU）與付費轉換。

### 寶箱等級一覽

| 等級 | 等待時間 | 保證最低幣值 | 額外獎勵（May Win） |
|------|---------|------------|-------------------|
| 🟠 Common | 1 小時 | TBD | 寶石、拼圖碎片 |
| 🔵 Rare | 2 小時 | TBD | Ball Shots、拼圖碎片 |
| 🟣 Epic | 4 小時 | TBD | Ball Shots、拼圖碎片 |
| 🔴 Legendary | 8 小時 | TBD | Ball Shots、拼圖碎片 |

### 核心規則

- 同時只能啟動 **1 個**寶箱計時
- 最多持有 **4 個**寶箱（滿格時仍可獲得，可選擇 PASS 或寶石即開）
- 可用**寶石**即時開啟，剩餘時間越長花費越多
- 觸發條件：單次贏分 ≥ Total Bet × **N**（N 值 TBD）
